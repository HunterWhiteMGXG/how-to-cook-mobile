# HowToCook Mobile

基于 [HowToCook](https://github.com/Anduin2017/HowToCook) 的移动端应用，使用 Taro 开发。

## 功能

- 菜谱浏览和搜索
- 烹饪模式（Tinder 风格卡片滑动）
- 收藏功能
- 远程数据更新

## 技术栈

- Taro 3.6
- React
- TypeScript
- Zustand
- SCSS

## 开发

```bash
cd app
npm install
npm run dev:h5
```

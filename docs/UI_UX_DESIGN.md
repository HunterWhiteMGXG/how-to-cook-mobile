# HowToCook Mobile App - UI/UX 设计方案

> 卡片式交互设计 + 厨房友好体验
> 创建日期：2025-11-02

---

## 🎨 设计理念

### 核心设计原则

1. **简洁明了** - 信息清晰，操作直观
2. **厨房友好** - 大按钮，易操作，防误触
3. **步骤导向** - 强调做菜流程，而非装饰
4. **效率优先** - 减少操作步骤，快速完成任务

### 设计灵感
- **卡片式交互** - Instagram Stories / Tinder
- **步骤流程** - 导航应用的分步指引
- **厨房场景** - 考虑湿手、油污场景

### 4. 卡片式做菜模式（Cooking Mode）⭐⭐⭐

这是整个应用的**核心创新功能**！

#### 全屏卡片设计
```
┌─────────────────────────────┐
│  ⬅  步骤 2/8        ☰       │ 顶部状态栏
├─────────────────────────────┤
│                             │
│                             │
│      [步骤示意图]           │ 大图展示
│                             │
│                             │
├─────────────────────────────┤
│  2. 切配                    │ 步骤标题
│                             │
│  将西红柿切成边长不超过      │
│  4cm 的小块                 │ 步骤内容
│                             │
│  💡 提示：                  │
│  可以去掉西红柿外皮          │ 提示信息
│                             │
│  ⏱ 预计 3 分钟              │ 时长
│                             │
├─────────────────────────────┤
│                             │
│  [        ✓ 完成，下一步     ] │ 主要操作按钮
│                             │
└─────────────────────────────┘
```

#### 手势交互
```
← 左滑：上一步
→ 右滑：下一步
↑ 上滑：显示完整菜谱
↓ 下滑：返回详情页
双击图片：放大查看
长按：添加计时器
```

#### 特殊功能

**1. 进度指示器**
```
● ● ○ ○ ○ ○ ○ ○
已完成  当前  未完成
```

**2. 计时器悬浮球**
```
┌─────────┐
│ ⏱ 3:45  │ 可拖动
└─────────┘
```

**3. 防息屏提示**
```
🔋 屏幕保持常亮
```

#### 组件清单
- **StepCard** - 步骤卡片（核心）
- **ProgressIndicator** - 进度指示器
- **FloatingTimer** - 悬浮计时器
- **GestureDetector** - 手势检测

#### 交互细节

**进入动画：**
```
详情页 → 淡出
  ↓
卡片式 → 放大进入
```

**切换动画：**
```
当前卡片 ← 滑动 → 下一卡片
  (100ms transition)
```

**完成步骤：**
```
点击"完成" → 打钩动画 → 自动跳转下一步
  (300ms 延迟)
```

**退出确认：**
```
点击返回 → 弹窗确认
  "确定要退出吗？进度将不会保存"
  [继续做菜] [确认退出]
```

## 🎯 交互动画

### 页面转场
```css
/* 右进 */
.slide-right-enter {
  transform: translateX(100%);
}
.slide-right-enter-active {
  transition: transform 300ms ease-out;
}

/* 淡入淡出 */
.fade-enter {
  opacity: 0;
}
.fade-enter-active {
  transition: opacity 200ms ease-in;
}
```

### 点击反馈
```css
/* 按钮点击 */
.button:active {
  transform: scale(0.95);
  transition: transform 100ms;
}

/* 卡片点击 */
.card:active {
  box-shadow: var(--shadow-md);
  transition: box-shadow 100ms;
}
```

### 加载动画
```css
/* 骨架屏 */
.skeleton {
  background: linear-gradient(
    90deg,
    #f0f0f0 25%,
    #e0e0e0 50%,
    #f0f0f0 75%
  );

```

## ♿ 无障碍设计

### 触摸目标
- 最小点击区域：44px × 44px
- 按钮之间间距：至少 8px

### 对比度
- 文本与背景对比度 ≥ 4.5:1
- 大文本对比度 ≥ 3:1

### 语义化
- 使用语义化标签
- 提供 aria-label
- 支持键盘导航

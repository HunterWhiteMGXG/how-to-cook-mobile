# 小程序品牌素材

## 名称建议

`爱学做菜`

## 小程序简介

面向厨房新手的家常菜谱与烹饪知识指南。

## 功能介绍

不知道今天吃什么？爱学做菜收录家常菜谱、食材用量和分步做法，支持分类浏览、搜索、收藏与烹饪模式，并提供实用的厨房基础知识，让第一次下厨也能按步骤完成一道菜。

## 审核说明

本小程序无需登录。进入首页后可按分类浏览菜谱，查看食材、用量和制作步骤；收藏数据仅保存在用户本机。理论栏目提供基础烹饪知识，不涉及用户发布内容、交易或付费服务。
